# stockmanage
Gestion des stocks


## 使用 Cordova 编译 stock

### 准备工作
1. 安装 Android SDK
2. JDK
3. Node.js
4. Cordova

### 创建应用 StockApp 并编译

cd D:\work\cs\c++\vs2015\cordova\hello
cordova create StockApp
cd StockApp
cordova platform add android
cordova build


编译成功后得到的 app-debug.apk 文件位于
D:\work\cs\c++\vs2015\cordova\StockApp\platforms\android\app\build\outputs\apk\debug
